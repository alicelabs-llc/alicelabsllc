# AliceLabs LLC

**Software & Technology Services — Wyoming, USA**

This repository is the home for AliceLabs LLC's company profile, internal templates, and onboarding documentation.

## What we build

- **AI Agents** — autonomous agents for GovTech, legal, and business workflows
- **GovTech** — federal contracting tools built on SAM.gov data
- **SaaS / Backend** — multi-tenant infrastructure with Supabase and TypeScript
- **Security Research** — bug bounty tooling, vulnerability research
- **LegalTech** — machine-readable legal datasets and AI-powered legal assistants
