## About

AliceLabs is a Wyoming-registered software and technology services company building products at the intersection of **artificial intelligence**, **cloud infrastructure**, and **federal technology**.

We design, develop, and deploy production-grade software — from SaaS platforms and mobile apps to AI-powered agents and automation pipelines.

---

## What We Build

| Area | Focus |
|---|---|
| 🤖 **AI & Agents** | LLM-powered tools, autonomous agents, Gemini / Claude integrations |
| 🌐 **SaaS Platforms** | Full-stack web applications with React, TypeScript, Supabase |
| 🏛️ **GovTech** | Federal contracting intelligence, SAM.gov automation |
| 📱 **Mobile** | Android apps (Kotlin / Jetpack Compose), cross-platform solutions |
| 🔐 **Security** | Bug bounty research, penetration testing, vulnerability analysis |
| ⚙️ **Automations** | CI/CD pipelines, data processing, API integrations |

## Products

| Product | Description | Stack |
|---|---|---|
| **OpenSAM** | Autonomous AI agent for federal contracting — SAM.gov discovery + Gemini analysis | React · Supabase · Gemini |
| **LexIA** | Legal assistant for Ecuadorian lawyers powered by Claude API | Claude API · TypeScript |
| **Bug Bounty Companion** | Android app for managing security research findings | Kotlin · Jetpack Compose |
| **Nexus CRM** | Customer relationship management for small businesses | React · PostgreSQL |
| **ScanTaxes** | Automated tax document processing | OCR · AI · Node.js |
